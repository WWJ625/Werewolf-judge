# 狼人杀法官助手

单文件 HTML 狼人杀法官助手，纯前端无依赖，直接在浏览器打开使用。

## 功能介绍

### 对局设置
- **9人局 / 12人局** — 两种模式自由切换
- **警徽规则** — 12人局可选，狼人自爆后警徽移交/撕毁
- **狼王类型** — 12人局可选狼王（死亡开枪）或白狼王（自爆带人）
- **守卫/愚者** — 12人局可选守卫生效或愚者判定

### 身份分配
- 自动随机分配身份，支持手动点击切换
- 未分配号码自动变为平民
- 每位玩家点击查看自己身份（号码牌翻转效果）

### 夜晚阶段
执行顺序：**狼人 → 预言家 → 女巫 → 守卫**

- **狼人** — 选择击杀目标，支持狼人自爆（普通狼人可选）
- **白狼王** — 专属自爆按钮，白天自爆带走一名玩家
- **预言家** — 查验一名玩家身份
- **女巫** — 解药/毒药各一瓶，第一夜可自救，之后不可
- **守卫** — 守护一名玩家，不可连续两晚守护同一人

### 白天阶段
- 自由讨论后投票放逐
- 狼王死亡时可开枪带走一人
- 狼人自爆后直接进入黑夜

### 胜负判定
- 狼人全灭 → 好人胜利
- 神职全灭或平民全灭（屠边）→ 狼人胜利
- 支持预判屠边（狼人选杀后无解药/无守卫时提前判定）

### 对局记录
- 每局结束后自动保存（sessionStorage，最多10条）
- 支持翻看历史对局的完整行动日志
- 刷新页面不丢失，关闭标签页自动清除
